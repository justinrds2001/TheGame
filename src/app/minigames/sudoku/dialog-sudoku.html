Sudoku Challenge

Objective:

The objective of the Sudoku AI Challenge is to showcase the remarkable power of artificial intelligence in solving complex puzzles. In this game, you will witness the capabilities of an AI algorithm that can swiftly solve Sudoku puzzles, providing an unparalleled demonstration of its computational abilities.

Rules:

  1. The puzzle is played on a 9x9 grid, divided into nine 3x3 subgrids called "boxes."
  2. The objective is to fill in the empty cells with digits from 1 to 9, such that each digit appears only once in each row, each column, and each box.
  3. Each row must contain all the digits from 1 to 9, without repetition.
  4. Each column must contain all the digits from 1 to 9, without repetition.
  5. Each 3x3 box must contain all the digits from 1 to 9, without repetition.
  6. Initially, some cells are pre-filled with numbers, and these clues should be used to deduce the values of the remaining empty cells.
  7. The puzzle is solved when all the empty cells are filled with valid numbers, following the above rules.
  8. There should be only one solution to a well-formed Sudoku puzzle.
