Mini-GPT4 Picture Describe Challenge

Objective:

The objective of the Mini-GPT4 Picture Describe Challenge is to test your visual recognition and describing skills by viewing an image and describing this image. Your description will be compared to a description written by the Mini-GPT4 AI who has viewed the same image. The goal of this minigame is to show you, the player, to what extend AI can understand images and describe what is happening in them.

Rules:

  1. Each round of the game consists of an image shown to the player.
  2. You will have an unlimited amount of time to write a description about this image which contains as much information about the image as possible.
  3. The description should be written in the player input area.
  4. Once the description is finished, press submit and your description will compared to a description generated by the Mini-GPT4 model based on the same image.
  5. The Mini-GPT4 generated description will be presented alongside your drawing for comparison.
  6. The comparison should be based on accuracy, precision and overall similarity between your description and the Mini-GPT4 description.
  7. Players can engage in multiple rounds with different images to improve their visual recognition skills and challenge themselves.

Tips:

  1. Pay attention to the details of the image.
  2. Try to avoid describing the same thing twice.
  3. Try to work in one direction (e.g. left to right, top to bottom) to avoid duplication.
  4. Don't worry about perfection. The focus is on expressing your visual interpretation of the image and seeing the difference with A.I.
  5. Analyze the Mini-GPT4 generated description and your description side by side to understand the similarities and differences.
  6. Learn from each round and strive to enhance your skills and understanding of A.I.
  7. Have fun and enjoy testing your visual recognizition skills!
